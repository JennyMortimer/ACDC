ACDC Database

The ACDC database is available here.

We welcome new submissions — please see the new clone submission page for further information. Please submit all data associated with the clone prior to shipping or dropping off. More information can be found in the FAQs.

Note: all clones in the ACDC collection were collected either from state land, or from private land with the owner's permission. Genus and species were determined by PCR barcoding (Borisjuk et al. 2014). Where barcoding has not given a clear species identity, this is noted, and we are in the process of whole-genome sequencing to confirm those few unknowns.
